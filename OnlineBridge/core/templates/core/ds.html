Datenschutz

Folgende personenbezogene Daten werden gespeichert:

  • Name
  • Anrede
  • Gehashtes Passwort
  • Anmeldung zu Turnieren

Das Passwort ist zu keinem Zeitpunkt in Klartext ersichtlich und kann daher nur mit den dafür vorgesehenen Routinen zurückgesetzt werden.

Es werden Anmeldungen an Turnierleiter und Clubs weitergegeben, ansonsten erfolgt keine Weitergabe von Daten.

Die Daten liegen auf einem Server innerhalb der Europäischen Union. Hier das Statement des Betreibers.
